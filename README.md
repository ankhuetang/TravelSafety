# Travel Safety Map

Travel Safety Map is a web application that provides users with information on the safety level, crime and traffic accidents to support users to have safe travelling experiences.

## Contributors:
- Khue Tang
- Phuong Cao
- Ngoc Doan
- Cuong Nguyen  

## Features
- Users can search for a location and get information on the safety level and traffic accidents around the area.
- Different colors are used to show safety level
- Warning messages will be sent when the subscribed location is below the standard safety rate.

## Technologies Used
- MongoDB
- Express.js
- React
- Node.js
- Twilio
- Cron job
- Web scraping
